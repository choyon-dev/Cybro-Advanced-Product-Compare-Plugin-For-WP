== Description ==

Cybro Product Compare for WooCommerce enables your customers to compare products side by side, facilitating informed purchasing decisions with ease.

= Key Features =

* Compare up to 4 products simultaneously
* AJAX-powered comparison (no page reload)
* Responsive and mobile-friendly comparison table
* Print and share functionality
* Customizable button placement
* Translation ready

= Usage =

1. Use the shortcode `[cybro_compare_table]` to display the comparison table anywhere on your site.
2. Use the shortcode `[cybro-compare]` to add a compare button to your product pages.
3. Configure button placement in WooCommerce → Product Compare settings.
4. Select which attributes to compare.
5. Choose your preferred table style.

= Pro Features =

* Unlimited product comparisons
* Email comparison table
* Custom attributes
* Advanced styling options
* Priority support

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/cybro-product-compare`.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to WooCommerce → Product Compare to configure settings.

= Minimum Requirements =

* WordPress 5.8 or greater
* WooCommerce 5.0 or greater
* PHP version 7.4 or greater

== Frequently Asked Questions ==

= How many products can be compared? =

The free version allows comparison of up to 4 products simultaneously.

= Is it compatible with my theme? =

Yes, the plugin is designed to work with any properly coded WordPress theme.

= Can I customize the comparison table? =

Yes, you can select which attributes to compare and choose from different table styles.

= Does it work on mobile devices? =

Yes, the comparison table is fully responsive and mobile-friendly.
